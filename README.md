# resharper-dark-theme

**ReSharper Dark Theme** is a Visual Studio Code (VS Code) extension that provides ReSharper Dark Theme.

## Installation

This extension is distributed via GitHub Releases. You can install it manually using the `.vsix` file.

1. Go to the [Releases](../../releases) page of this repository.
2. Download the latest `.vsix` file (e.g., `resharper-dark-theme-1.0.0.vsix`).
3. Open Visual Studio Code.
4. Open the **Extensions** view (`Ctrl+Shift+X` on Windows/Linux, `Cmd+Shift+X` on macOS).
5. Click on the `...` (Views and More Actions) icon in the top right corner of the Extensions view.
6. Select **Install from VSIX...** from the dropdown menu.
7. Choose the `.vsix` file you downloaded.

*Alternatively, you can open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`), type `Extensions: Install from VSIX...`, and select your downloaded file.*
